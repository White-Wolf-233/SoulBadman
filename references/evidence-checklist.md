# 证据收集操作指南

> 不要只说"收集证据"——告诉伙计每一步怎么操作。

## 证据的用途

| 证据类型 | 证明什么 | 关键程度 |
|---------|---------|---------|
| 劳动合同 | 劳动关系、工资标准、合同期限 | ⭐⭐⭐ |
| 工资记录 | 工资标准（赔偿计算基数） | ⭐⭐⭐ |
| 解除通知 | 辞退事实、辞退理由 | ⭐⭐⭐ |
| 考勤记录 | 加班事实 | ⭐⭐ |
| 沟通记录 | HR 的辞退理由、威胁、诱导自辞 | ⭐⭐⭐ |
| 工作群聊天 | 劳动关系、工作安排 | ⭐⭐ |
| 绩效记录 | 反驳"不胜任" | ⭐⭐ |

---

## P0：立刻做（离职前可能仅有一次机会）

### 1. 导出考勤记录

```
钉钉 → 工作台 → 考勤打卡 → 统计 → 选择日期范围 → 截图
企业微信 → 工作台 → 打卡 → 月报 → 截图
飞书 → 工作台 → 考勤 → 导出报表（PDF）
```

导出最近 1-2 年的全部考勤记录。**尤其是加班日。**

### 2. 截图组织架构

```
钉钉 → 通讯录 → 找到自己 → 截图（显示你的部门和上级）
企业微信 → 通讯录 → 组织架构 → 找到自己 → 截图
```

截图要包含你的名字 + 部门名称。这证明你是这家公司的员工。

### 3. 导出工作邮件

```
打开邮箱 → 搜索关键词逐个导出：
  - "绩效" → 导出全部含绩效考核的邮件
  - "解除" / "辞退" / "协商" → 导出全部与 HR 沟通的邮件
  - 搜索你的直属领导的名字 → 导出工作安排邮件
  - 导出近 3 个月所有已发送邮件
```

**技巧**：使用邮箱的"全部导出"功能或逐个截图。离职后公司会注销你的邮箱。

### 4. 保存工资记录

```
方式一（最可靠）：打开银行 APP → 搜索"工资" → 截图最近 12 个月的工资入账记录
方式二：个税 APP → 收入纳税明细 → 截图
方式三：公司内部的薪资系统 → 截图
```

**注意**：截图要显示"工资"字样或固定的入账金额。银行流水是最硬的证据。

### 5. 截图 HR 聊天记录

```
钉钉/企业微信/飞书 → 找到和 HR 的聊天记录 → 把关于辞退的部分逐屏截图
→ 从第一条相关消息截到最后一条
→ 截图时要包含日期时间
```

**不要删除聊天记录。** 截图的同时保留原始记录。

## P1：尽快做（趁还在职）

### 6. 合法录音

**什么时候录**：下次和 HR 或领导谈话时。

**怎么录**：
- 手机打开录音 APP，开始录音
- 放桌上或口袋里（合法：正常办公场所的面对面谈话或电话）
- 用引导式提问让对方说出关键信息

**引导式提问模板**：
> "那您的意思是，公司因为我绩效不达标所以要解除劳动合同，对吗？"
> "您说架构调整——具体是哪个部门合并了？"
> "如果我不同意这个方案，公司下一步会怎么做？"

**不要**：
- 窃听、偷拍私人空间（违法）
- 诱导对方撒谎
- 剪辑录音（提交时保持完整）

### 7. 下载绩效记录

```
公司绩效系统 → 找到历年绩效评估 → 逐一截图
保存内容包括：
  - 每季/每年的绩效考核结果（评分、评级）
  - 绩效面谈记录
  - 目标设定 OKR/KPI
  - 如果有 PIP（绩效改进计划），一定截图
```

**为什么重要**：如果公司说你不胜任，但绩效记录显示你一直是正常/B级以上——这就是最有力的反驳证据。

### 8. 保存加班证据

```
钉钉/企业微信 → 审批 → 加班审批 → 截图
邮件 → 搜索"加班" → 导出
工作群 → 搜索"加班"/"周末"/"上线" → 截图
打车记录 → 滴滴/高德 → 搜索下班时间的打车记录 → 截图（证明下班时间）
```

## P2：其他证据

### 9. 劳动合同

拍照或扫描整份劳动合同（每一页）。确保拍到：
- 合同期限
- 工资条款
- 岗位名称

如果公司没给你合同 → 这本身就是违法行为，触发双倍工资。

### 10. 工牌 / 门禁卡

拍照保存。

### 11. 入职登记表 / Offer Letter

拍照保存。上面有入职日期和约定的薪资。

### 12. 公司制度文件

如果有员工手册、绩效考核制度等内部文件，拍照保存相关页面。

## 整理方式

把以上所有证据按文件夹整理：

```
我的维权证据/
├── 01-劳动关系/
│   ├── 劳动合同.pdf
│   ├── 组织架构截图.png
│   ├── 工牌照片.jpg
│   └── offer-letter.pdf
├── 02-工资记录/
│   ├── 银行流水_2024年1-12月.png
│   └── 个税记录.png
├── 03-解除相关/
│   ├── HR聊天记录_全部截图/
│   ├── 解除通知（如有）.pdf
│   └── 录音_2024年X月X日.mp3
├── 04-绩效记录/
│   ├── 2023年度绩效.png
│   └── 2024年Q1-Q3绩效.png
├── 05-考勤加班/
│   ├── 考勤记录_2024年.pdf
│   └── 加班审批截图.png
└── 06-其他/
    ├── 员工手册_相关页.jpg
    └── 年假余额截图.png
```
