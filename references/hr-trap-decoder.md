# HR 话术解码表

> HR 说的每句话，都有法律上的含义。知道他们真正在说什么，才能反击。

## 核心原则

HR 的工作目标：**以最低成本让公司合法（或不合法）地解除你的劳动合同。**

你的目标：**最大化赔偿收益，不让 HR 钻空子。**

---

## 话术解码

### 「公司架构调整，你的岗位取消了」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 用第 40 条（客观情况变化）以 N+1 的成本开掉你 |
| **法律含义** | 公司需要证明：①客观情况真的变了 ②已经和你协商过调岗 ③没有合适的岗位 |
| **你的漏洞** | 如果公司拿不出证据，或者没提供过调岗机会 → 违法解除 → 2N |
| **你的回应** | 「公司有给我提供其他岗位选择吗？具体是哪些岗位？」「请公司出具正式的解除通知，注明解除理由。」 |
| **战术** | 逼 HR 书面化——如果 HR 不敢把"架构调整"写进正式通知，说明他们自己也心虚。 |

### 「你的绩效不达标，不胜任这份工作」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 用第 40 条（不胜任）N+1 开掉你 |
| **法律含义** | 公司必须证明：①给过你培训或调岗 ②培训/调岗后仍然不胜任 |
| **你的漏洞** | 你有 PIP 书面记录吗？公司安排过培训吗？最近绩效评估是多少分？ |
| **你的回应** | 「我最近一次绩效改进计划（PIP）是什么时候？有书面记录吗？」「公司在解除之前安排过哪些培训或调岗？」 |
| **战术** | 绝大多数公司根本没有走完 PIP 流程。拿不出书面证据 = 违法解除。 |

### 「建议你主动离职吧，对你下一份工作背调好」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 让你自辞，公司一分不赔 |
| **法律含义** | 主动辞职 = 第 37 条 = 零赔偿 |
| **你的回应** | 「我不主动辞职。请公司按法律规定的程序处理。」 |
| **战术** | 这是最经典的 HR 骗术。记住三字诀——**不自辞**。你一旦主动提离职，几万到十几万的赔偿金就没了。 |

### 「末位淘汰，你是最后一名」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 用排名淘汰你但不走法律程序 |
| **法律含义** | 最高人民法院明确说了：末位淘汰不等于不胜任。末位淘汰直接解除 = 违法解除 |
| **你的回应** | 「排名最后不等于不胜任这份工作。公司要我走可以，请按法律规定的赔偿标准来。」 |
| **战术** | 末位淘汰解除劳动合同的违法性认定在司法实践中已有较明确支持倾向（最高法指导案例第18号），劳动者胜诉风险较低，但仍取决于证据充分性和当地裁判口径。 |

### 「签了这个拿钱走人，不签一分都没有」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 恐吓你接受低于法定标准的方案 |
| **法律含义** | 不签字公司也不能不给你法定的赔偿。威胁不给钱是虚张声势 |
| **你的回应** | 「我需要时间考虑。您方便把这个方案发我邮件吗？」 |
| **战术** | 要求 HR 把方案发邮件——如果方案本身低于法定标准，这就是证据。永远不要在当场签任何东西。 |

### 「试用期不合格」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 在试用期低成本开掉你 |
| **法律含义** | 公司必须证明：①录用时就明确了录用条件 ②告知过你 ③你确实不符合 |
| **你的回应** | 「录用时告知过我不符合哪些条件吗？有书面记录吗？」 |
| **战术** | 绝大多数公司在试用期辞退拿不出录用条件证明 = 违法解除 = 2N。 |

### 「公司最近困难，希望大家理解」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 道德绑架，让你接受低赔偿 |
| **法律含义** | 公司困难不等于法律不保护你。经济补偿金在清算顺序上排在第一梯队 |
| **你的回应** | 「我理解。请公司出具正式的解除通知。」 |
| **战术** | 不用跟 HR 掰扯感情。法律的事，谈法律。 |

### 「不签协商解除，我们就走单方解除，钱更少」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 逼你签字 |
| **法律含义** | 单方解除如果合法 → N+1；如果违法 → 2N。通常比协商解除（可能谈更高的条件）的结果更好 |
| **你的回应** | 「公司如果认为可以单方解除，请出正式通知。我收到后依法应对。」 |
| **战术** | 不要被唬住。公司的单方解除如果站不住脚，2N 比协商拿得更多。 |

### 「这是公司的最终方案，不接受就没机会了」

| 项目 | 内容 |
|------|------|
| **HR 的真实意图** | 制造紧迫感 |
| **法律含义** | 仲裁时效是 1 年。不存在"不接受就没了"的说法 |
| **你的回应** | 「我需要 24 小时考虑。明天回复你。」 |
| **战术** | 任何时候都不要当场做决定。HR 越急，说明越有问题。 |

---

## 万能回应模板

以下四个回应适用于大部分 HR 谈话场景：

1. **「请公司出具盖公章的正式书面通知。」**
   → HR 如果不敢出 = 他们自己也不确定是否合法

2. **「我不主动辞职。请公司按法律规定的程序处理。」**
   → 封死"劝自辞"路线

3. **「我需要时间考虑。您方便把这个方案发我邮件吗？」**
   → 避免当场决定 + 留下书面证据

4. **「我了解了。在收到正式通知前，我继续正常上班。」**
   → 不给 HR"旷工"的口实
